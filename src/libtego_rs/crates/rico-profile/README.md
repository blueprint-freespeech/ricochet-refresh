# rico-profile

This is a rust crate used for reading+writing Ricochet-Refresh user profile files.

## Features

- `v3-profile` : enables reading legacy Ricochet-Refresh v3.0.X series JSON-encoded profiles
- `v4-profile` : enbales reading+writing Ricochet-Refresh v4+ series SQLCipher-encoded encrypted profiles

## Specifications

- **Legacy JSON Profile**: https://github.com/blueprint-freespeech/ricochet-refresh/blob/main/doc/settings-v3.md
- **Encrypted Profile**: https://github.com/blueprint-freespeech/ricochet-refresh/blob/main/doc/profile-v4.md
